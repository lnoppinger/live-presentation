Packages

- Bündeln von Klassen zur Strukturierung, Austauschbarkeit, Wiederverwendbarkeit, getrenntes entwickeln / testen - Einzelne Methoden: package.klasse.methode() - ganze Klasse: import package.klasse ganz oben - ganes Package: import package.* - Häufig verwendete Packages: java.io, java.util, java.lang - Eigene Packages mit package packagename oben in jeder Klasse

Sichtbarkeiten

public: überall nichts (package scope): überall im Package protected: überall im Package und in Unterklassen (Package egal) private: nur selbe Datei
