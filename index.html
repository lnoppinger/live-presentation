Now to the additional navigation features: 

            Space    -> next slide
            Backspace-> previous slide
            A        -> add slide
            D        -> add slide with canvas
            E        -> edit slide
            H        -> display help
            M        -> enable presenter mode
            N        -> show notes
            P        -> print
            Q        -> return to current slide
            R        -> reload slide
            S        -> download html
            Ctrl +   -> zoom in
            Ctrl -   -> zoom out

Now it's your turn to create a presentation.
